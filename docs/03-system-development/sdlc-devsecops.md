# 개발 프로세스 통제
**SDLC & DevSecOps Controls**

:::info 관련 표준
CISA Domain 3.2 / OWASP SAMM / NIST SP 800-218 / ISO/IEC 27034
:::

:::note 작성 예정
이 문서는 작성 중입니다.
:::

## 주요 토픽
- 요구사항 정의 및 보안 요구사항 검토
- 소스코드 보안 검토(SAST/DAST/SCA) 기준
- CI/CD 파이프라인 내 통제 식별 및 자동화 테스트
- DevSecOps 보안 내재화(Shift-Left Security)
- 코드 리뷰 및 취약점 관리 프로세스
