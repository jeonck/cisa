# RCM (Risk Control Matrix) 라이브러리
**Risk Control Matrix Library**

:::info 활용 안내
각 문서는 `[관련 표준: CISA Domain X.X / ISO 27001 Control X.X]` 형태로 상호 참조 태그를 포함합니다.
:::

:::note 작성 예정
이 문서는 작성 중입니다.
:::

## RCM 구조

| 항목 | 설명 |
|------|------|
| **위험(Risk)** | 식별된 위험 사건 및 위험 설명 |
| **통제 목적** | 위험 완화를 위한 통제 목적 |
| **통제 활동** | 예방/탐지 통제 활동 설명 |
| **통제 유형** | 수동/자동, 예방/탐지/교정 |
| **테스트 절차** | 감사인 수행 테스트 절차 |
| **증거 유형** | 수집 증거 유형 |

## 프로세스별 RCM 목록
- IT 일반 통제(ITGC) RCM
- 애플리케이션 통제(AC) RCM
- 접근 관리(Access Management) RCM
- 변경 관리(Change Management) RCM
- 운영 및 백업(Operations & Backup) RCM
