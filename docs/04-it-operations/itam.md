# IT 자산 및 구성 관리
**IT Asset Management & Configuration (ITAM/CMDB)**

:::info 관련 표준
CISA Domain 4.1 / ITIL v4 / ISO/IEC 19770
:::

:::note 작성 예정
이 문서는 작성 중입니다.
:::

## 주요 토픽
- 하드웨어/소프트웨어 자산 인벤토리(Asset Register) 관리 기준
- CMDB(Configuration Management Database) 무결성 감사
- 라이선스 컴플라이언스(SAM) 검증
- 자산 폐기(Disposal) 프로세스 및 데이터 완전 삭제
