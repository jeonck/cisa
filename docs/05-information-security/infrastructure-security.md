## 주요 토픽
- 네트워크 세분화(Segmentation) 및 Zero Trust 아키텍처
- 방화벽/IDS/IPS 룰셋 감사 기준
- OS/서버 보안 설정(Hardening Baseline) — CIS Benchmark
- 취약점 진단(VA) 및 침투 테스트(PT) 연계
- DMZ 구성 및 인터넷 노출 면적 최소화
