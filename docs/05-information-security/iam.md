# ID 및 접근 관리
**Identity & Access Management (IAM)**

:::info 관련 표준
CISA Domain 5.1 / ISO/IEC 27001 A.9 / NIST SP 800-63 / CIS Control 5-6
:::

:::note 작성 예정
이 문서는 작성 중입니다.
:::

## 주요 토픽
- 사용자 계정 수명주기(생성-변경-삭제-재검토) 관리
- 권한 최소화(Least Privilege) 원칙 및 Need-to-Know
- 특권 계정(PAM: Privileged Access Management) 모니터링
- 감사 추적(Audit Trail) 및 접근 로그 검토
- MFA(다중 인증) 구현 및 검증
- SoD 위반 탐지 자동화
