# 침해사고 대응 및 디지털 포렌식
**Incident Response & Digital Forensics**

:::info 관련 표준
CISA Domain 5.5 / NIST SP 800-61 / ISO/IEC 27035 / RFC 3227
:::

:::note 작성 예정
이 문서는 작성 중입니다.
:::

## 주요 토픽
- 보안 이벤트 모니터링(SIEM/SOAR) 구현 및 감사
- 위협 인텔리전스(CTI) 활용 방안
- 침해사고 대응(IR) 6단계 — 준비, 탐지, 격리, 근절, 복구, 교훈
- 증거 수집 체인(Chain of Custody) 유지 절차
- 디지털 포렌식 기법 및 법적 증거 요건
