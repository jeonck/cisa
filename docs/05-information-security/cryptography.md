## 주요 토픽
- 데이터 저장 시(At Rest) 암호화 표준 — AES-256
- 전송 시(In Transit) 암호화 표준 — TLS 1.3
- 키 관리(Key Management) 수명주기 및 HSM 활용
- 인증서 관리(Certificate Lifecycle Management)
- 양자 내성 암호화(Post-Quantum Cryptography) 동향
